## 2. GPU/CPU双方案设计

### 2.1 方案对比

| 方案 | 训练模式 | 推理模式 | 显存要求 | 内存要求 | 训练时间(500条) | 适用场景 |
|------|----------|----------|----------|----------|-----------------|----------|
| GPU方案 | GPU训练 | GPU推理 | ≥8GB | ≥16GB | 2-4小时 | 有GPU环境 |
| CPU方案 | CPU训练 | CPU推理 | 无 | ≥16GB | 12-24小时 | 无GPU环境 |
| QLoRA方案 | GPU训练(量化) | GPU推理 | ≥4GB | ≥12GB | 4-8小时 | 显存受限 |

### 2.2 自动检测机制

```python
def detect_hardware():
    if torch.cuda.is_available():
        return "GPU"  # 使用GPU训练+推理
    elif psutil.virtual_memory().total >= 16GB:
        return "CPU"  # 使用CPU训练+推理
    else:
        raise Exception("内存不足，至少需要16GB内存")
```

### 2.3 硬件配置要求

**GPU方案（推荐）**
- 最低: NVIDIA RTX 3060 (8GB显存) / RTX 4090 (24GB显存)
- CPU: i5-12700H+
- 内存: 16GB+
- 存储: 20GB+

**CPU方案**
- CPU: i5-10400+
- 内存: 16GB+ (建议32GB)
- 存储: 20GB+
- 训练时间较长，但可正常运行

## 8. 技术亮点（简历用）

- **LoRA轻量微调**：仅调整模型0.4%参数，实现轻量化训练
- **QLoRA量化训练**：4-bit精度量化，降低显存占用至原来的1/4
- **领域自适应**：基于行业配对数据微调，模型具备行业专家能力
- **低秩矩阵原理**：通过低秩矩阵分解替代全参数更新
- **本地部署保障数据安全**：纯本地运行，无云端依赖
